För att logga in

För att logga in i prototypen finns 5 st testkonton:

Användarnamn     Lösenord
student1 student1
student2 student2
student3 student3
student4 student4
student5 student5

Gå till http://ladok.herokuapp.com för att logga in

När du loggat in, tryck gärna på "Ändra inställningar" och byt till din mejladress. På detta sätt kan du testa att systemet skickar bekräftelsemejl till dig när du utför olika aktiviteter i systemet. OBS! Glöm inte att ändra "notifiera via mejl" till "ja".

_______________________________________

Skapa eget konto*

Om du vill kan du skapa ett eget konto och testa med. Tryck då på knappen "Skapa studentkonto" som finns till höger om knappen "logga in" på inloggningssidan.

*Observera att denna funktion endast finns för att kunna testa systemet. Normalt sett sker registrering av studentkonton hos studieadministrationen.
